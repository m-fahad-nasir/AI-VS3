## ✨ Key Contributions

This work introduces a novel temporal greenhouse tomato dataset (Tomato-VS3) and an AI-assisted plant health assessment framework termed AI-Assisted Vertical Stem Stratified Scouting (AI-VS3). The proposed dataset captures weekly temporal observations from greenhouse tomato stems across multiple stratified stem regions, including basal, medial, and apical leaves. Unlike conventional agricultural datasets, the collected data are annotated by expert agronomists with plant health status, disease category, precise disease, disease severity, confidence assessment, and evidence-based reasoning descriptions. Furthermore, this work establishes a multimodal benchmark for evaluating Vision-Language Models on fine-grained plant health assessment tasks, including disease classification, severity estimation, reasoning generation, and temporal disease progression analysis under realistic greenhouse conditions.

In addition, we propose a modular AI framework integrating leaf-level object detection, vision-language reasoning, and a hierarchical plant health scoring algorithm for structured plant-level risk assessment. The object detection module isolates relevant leaf regions to reduce environmental noise, while the VLM module performs multimodal disease reasoning and symptom interpretation. These outputs are subsequently integrated across multiple leaf images into a structured plant health scoring framework that incorporates disease severity, confidence estimation, vertical stem importance, and multi-image agreement-based aggregation for comprehensive plant-level analysis. Extensive experiments demonstrate the challenges of achieving expert-level plant disease reasoning using current VLMs, particularly for early-stage disease recognition and severity estimation.Overall, this work contributes Tomato-VS3, a novel temporal multimodal greenhouse tomato dataset, a benchmarking framework for greenhouse plant health analysis, and an explainable AI-assisted scouting methodology for advancing intelligent agricultural monitoring systems.
